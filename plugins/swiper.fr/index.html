    Outil de balayge

    Qu’est-ce que le module d’extension de l’outil de balayage ?

    Le module d’extension de l’outil de balayage offre une interface intuitive pour comparer facilement le contenu entre les couches. Comme indiqué à la Figure 45, l’utilisateur peut déplacer la barre de balayage horizontalement pour examiner attentivement les différences entre deux couches, qui pourraient être dues à un changement temporel. L’outil de balayage peut également être utilisé pour révéler le contenu d’une couche par rapport à une autre.

    Quand l'utiliser ?

    Si vous avez deux cartes qui représentent le même thème utilisant les mêmes méthodes de cartographie, mais pour des périodes différentes, vous pouvez utiliser le module d'extension de l’outil de balayage pour améliorer l’expérience de l’utilisateur dans l’exploration de tout changement au fil du temps et de l’espace. Par exemple, pour visualiser l’imagerie avant et après une inondation ou afficher deux couches thématiques connexes sur une carte, l’outil de balayage permet à l’utilisateur de déplacer le curseur d’un côté à l’autre pour voir les changements.

    Il est important de noter que dans cet exemple (Figure 45) les deux couches de couverture terrestre sont très semblables. Cependant, il y a une différence subtile qui peut être facilement détectée lors de l’utilisation l’outil de balayage.

    Astuce

    Pour trouver la couche active de l’outil de balayage, placer simplement le curseur de la souris sur la barre verticale.

    Conseil pour interagir avec le module d’extension de l’outil de balayage : Lors de l’utilisation de l’outil de balayage, la couche activée sera temporairement effacée par rapport à la position de la barre de l’outil de balayage. Il est recommandé que la couche active du module d’extension de l’outil de balayage soit la couche supérieure de l’arborescence de la légende.

    Comment l’utiliser – un cas d’utilisation

    Dans ce cas d’utilisation (Figure 46), le module d’extension de l’outil de balayage a été utilisé pour mettre en évidence les centrales hydroélectriques et pétrolières plutôt que pour détecter les changements. En activant le module d’extension sur les centrales hydroélectriques de 100 MW et 1 MW tout en déplaçant la barre de balayage, l’utilisateur peut clairement voir l’emplacement de différentes centrales électriques pour les régions d’intérêt. De plus amples renseignements sur la source des données peuvent être obtenus dans les métadonnées.

    Configuring the Swiper plugin

    Before you begin, Ensure that all Layers to be used should be configured in Authoring tool under the Map Tab. (See Section: Using the Authoring Tool to Configure a Plugin – Adding a new layer)

    Step 1. Enable Swiper

    Navigate to the Plugins section in the FGP Authoring Tool.

    Under the Swiper tab, click on Enable.

    By default. The “Type” will be set vertical. At the present time, this is the only option available. The “keyboard movement” is set to 10 by default. This is the recommended value; however, you may change it according to your use case.

    Step 2. Add Layers

    Under the Layer Id (Figure 47), ensure that the layer to be used with the Swiper plugin is selected. In this use case, the Layer Id is automatically generated after adding layers in the Authoring Tool. Selecting the “2015 Land Cover” Layer will result in the Swiper being enabled on that layer.

    Note

    The layer selected here will be the layer the Swiper plugin is enabled on.

    Definition of Parameters

    Listed in the following table are the definitions of all the parameters available when using the FGP Authoring tool to configure the plugin to work with an applicable map service.

    Parameters Description
    General Type Determines oi the swiper is vertical or horizontal. Note: Only vertical is supported at the moment
    Keyboard Offset Determines the number of pixels to move the swiper when the keyboard is used for navigation
    Layers Id Layer id as defined in the layer section
