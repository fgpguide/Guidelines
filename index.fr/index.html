Bienvenue dans Guidelines for FGP Plugins

Bienvenue dans le Guide des plugins développés pour la plateforme FGP. L'objectif de ces lignes directrices est d'encourager l'utilisation appropriée de ces plugins sur FGP pour améliorer la recherche, la visualisation, l'analyse et le développement d'applications.

Ces plugins sont créés pour offrir de nouvelles façons intéressantes de présenter vos riches ensembles de données aux utilisateurs. Les directives suivantes se concentreront sur la bonne initialisation des paramètres de configuration pour une intégration réussie des plugins. Une définition des paramètres, des structures de données attendues ainsi qu'un guide de configuration seront fournis en plus des cas d'utilisation interactifs utilisant des ensembles de données existants trouvés sur FGP.
